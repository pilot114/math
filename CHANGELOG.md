# Changelog

Все значимые изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-21

### Изменено
- Обновлены все зависимости до последних совместимых версий:
  - `vue`: 3.2.33 → 3.5.13
  - `mathjs`: 10.5.1 → 15.1.0 (major update!)
  - `chart.js`: 3.7.1 → 3.9.1
  - `core-js`: 3.22.3 → 3.47.0
  - `@vue/cli-plugin-babel`: 5.0.4 → 5.0.8
  - `@vue/cli-plugin-eslint`: 5.0.4 → 5.0.8
  - `@vue/cli-service`: 5.0.4 → 5.0.8
  - `@vue/compiler-sfc`: 3.0.0 → 3.5.13
  - `eslint`: 6.7.2 → 8.57.1 (major update!)
  - `eslint-plugin-vue`: 7.0.0 → 9.31.0 (major update!)

### Добавлено
- Заменён устаревший `babel-eslint` на `@babel/eslint-parser` 7.25.9
- Настроено правило ESLint для отключения требования многословных имён компонентов
- Создан детальный ROADMAP.md с планом развития библиотеки
- Обновлён README.md с документацией по использованию
- Создан CHANGELOG.md для отслеживания изменений

### Исправлено
- Исправлены проблемы безопасности в зависимостях (35 → 13 уязвимостей)
- Обновлены security-зависимости:
  - `ansi-html`: >=0.0.8 → >=0.0.9
  - `ansi-regex`: >=5.0.1 → >=6.1.0
  - `ejs`: >=3.1.7 → >=3.1.10
  - `glob-parent`: >=5.1.2 → >=6.0.2
  - `minimist`: >=1.2.6 → >=1.2.8
  - `node-forge`: >=1.3.0 → >=1.3.1
  - `nth-check`: >=2.0.1 → >=2.1.1
  - `url-parse`: >=1.5.9 → >=1.5.10
- Исправлена конфигурация ESLint для совместимости с новыми версиями

### Технические детали
- Проект успешно собирается с помощью `npm run build`
- ESLint проверка проходит без ошибок
- Требуется флаг `NODE_OPTIONS=--openssl-legacy-provider` для сборки на Node.js 22+

## [0.0.1] - Initial Release

### Добавлено
- Базовая визуализация функций одной переменной
- Анимация с параметром времени `t`
- Вычисление производных функций
- Композиция функций (функции могут использовать значения других функций)
- Интерактивное управление: play/pause, добавление/удаление функций
- Сохранение и загрузка конфигураций в localStorage
- Настройка минимума и максимума оси Y
- Поддержка библиотеки math.js для парсинга математических выражений
